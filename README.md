# Mini Backend

A minimal FastAPI backend with four endpoints, built to demonstrate the request → response loop from server side.

## Run it

```bash
pip install fastapi uvicorn
uvicorn main:app --reload --host 0.0.0.0
```

## Endpoints

| Method | Route | Description |
|--------|-------|-------------|
| GET | `/` | Health check with timestamp |
| GET | `/api/status` | Service status |
| GET | `/api/greet/{name}` | Greets a name, optional `?loud=true` |
| POST | `/api/echo` | Echoes back the JSON body sent |

## Sample outputs

**GET `/`**
```bash
$ curl http://localhost:8000/
{"message":"Server is alive","time":"2026-07-09T05:25:53.173360"}
```

**GET `/api/status`**
```bash
$ curl http://localhost:8000/api/status
{"status":"ok","service":"mini-backend","version":"1.0"}
```

**GET `/api/greet/{name}`**
```bash
$ curl http://localhost:8000/api/greet/jk?loud=true
{"message":"HELLO, JK!"}
```

**POST `/api/echo`**
```bash
$ curl -X POST http://localhost:8000/api/echo \
  -H "Content-Type: application/json" \
  -d '{"course": "backend101"}'
{"you_sent":{"course":"backend101"},"received_at":"2026-07-09T05:24:51.653894"}
```
